# PDF to Markdown Converter

A simple Python tool that converts PDF files into Markdown format using Microsoft's MarkItDown library.

## Features

- Converts PDF files into Markdown
- Simple command-line usage
- Uses Microsoft's MarkItDown library
##EXTRA ADD -Batch conversion , complete folder with many pdf converted into .md file

## Requirements

- Python 3.10 or higher
